# Layout JSON Index

This folder contains example JSON payloads for all supported layouts.

## Naming Convention

- Format: `NN-layout-key.json`
- `NN`: 2-digit order index (`01`, `02`, ...)
- `layout-key`: exact layout name used by `"layout"` in slide JSON

Do not rename existing files unless all references are updated together.
